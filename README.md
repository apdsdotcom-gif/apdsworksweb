# APDSWorks
Portofolio pastel modern (Next.js + Tailwind + Framer Motion).

## Jalankan lokal
npm i
npm run dev

## Deploy ke Vercel
- Buat project baru, import repo ini
- Domain: apdsworks.xyz (point ke Vercel)
- Pastikan gambar ada di /public/images/*

## Aset
SVG wave & blob di /public/svg/*
OG image default: /public/images/og-default.webp
Favicon: /public/favicon.svg
